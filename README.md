# Size of Life

An interactive visualization exploring the scale of objects in our universe, from subatomic particles to the observable universe.

## Features

- **Interactive Scale Exploration**: Smoothly zoom through 62 orders of magnitude
- **Visual Representation**: Objects drawn to scale with color-coded categories
- **Detailed Information**: Learn about each object as you explore
- **Responsive Design**: Works on desktop and mobile devices
- **Smooth Animations**: Fluid transitions between scales

## How to Use

1. **Zoom Controls**: Use the slider, +/- buttons, or mouse wheel to zoom in and out
2. **Keyboard Shortcuts**: 
   - Arrow Up / + : Zoom in
   - Arrow Down / - : Zoom out
3. **Touch Support**: Swipe up/down on mobile devices

## Objects Included

The visualization includes objects from multiple categories:
- Quantum particles (Planck length, protons, atoms)
- Molecular structures (DNA, viruses, bacteria)
- Microscopic life (cells, dust mites)
- Animals (from fleas to blue whales)
- Human-made structures (buildings, monuments)
- Planetary and cosmic objects (Earth, Sun, galaxies, universe)

## Technologies

- Pure JavaScript (no frameworks)
- HTML5 Canvas for rendering
- CSS3 for styling
- Logarithmic scale system for smooth navigation

## Run Locally

Simply open `index.html` in a modern web browser. No build process or server required.

## Live Demo

Open `index.html` directly in your browser or serve with any static file server:

```bash
python3 -m http.server 8000
# Then visit http://localhost:8000
```
